# Support

## Usage Questions

Use [GitHub Discussions](https://github.com/chokonaira/reddial/discussions) for setup questions, adapter design, rubric methodology, and sanitized report interpretation.

## Bugs And Feature Requests

Use the repository's structured issue forms for reproducible bugs and concrete feature proposals. Include the RedDial version, Node.js version, target type, command, expected result, and sanitized output.

Do not post API keys, authenticated endpoint URLs, private transcripts, or proprietary knowledge-base documents. Security vulnerabilities belong in [private vulnerability reporting](https://github.com/chokonaira/reddial/security/advisories/new).

RedDial is maintained on a best-effort basis and does not currently include guaranteed response times or commercial support.
